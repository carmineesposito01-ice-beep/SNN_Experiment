# opt_plots/ — Grafici di ottimizzatore

Archivio di grafici diagnostici prodotti durante gli studi sugli **ottimizzatori**
(Adam / Prodigy / EventProp): curve di loss, learning-rate, norma del gradiente, spike-rate,
confronti tra varianti.

Sono materiale di supporto/consultazione. I verdetti sintetici degli studi ottimizzatore sono
nei `.md` di `document/` (es. `PRODIGY_DEEP_STUDY.md`, `PRODIGY_STUDY_CLOSURE.md`) e i dati
grezzi in `results/Prodigy_Study/`.
