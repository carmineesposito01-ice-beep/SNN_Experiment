## Task 3: Parità `core-fixed == fsm` per champion (il gate di sicurezza)

**Files:** Create `matlab/run_b2_parity.m`

La certificazione che il datapath serializzato mirrora il core fixed, per champion. È il **gate**: nessun champion
si sintetizza senza 0 mismatch. Il core fixed di riferimento è quello di `run_fixed_sweep` (già gira su tutti i champion).

- [ ] **Step 1: scrivi l'harness di parità**
```matlab
function nmis = run_b2_parity(name)
%RUN_B2_PARITY  0 mismatch = il fsm B2 (ROM attiva) mirrora bit-exact il core fixed, per `name`.
  here = fileparts(mfilename('fullpath'));
  gen_b2_rom(name); rehash;                         % ROM attiva = champion `name`
  d = load(fullfile(here, 'champions_export.mat')); champs = d.champions;
  if iscell(champs), champs = [champs{:}]; end
  c = champs(find(arrayfun(@(x) strcmp(char(string(x.name)), name), champs), 1));
  T = snn_types('fixed', 13); W = b2_rom_active();
  N = size(c.x_phys, 1); nmis = 0;
  % riferimento: raw del core fixed, tick-by-tick (come snn_b2_fsm produce 'raw' a fine control-step)
  snn_core([], [], T, 'reset');
  for t = 1:N
    xn = cast(snn_normalize(c.x_phys(t,:).', c.norm), 'like', T.V);
    raw_core = snn_core(xn, to_weights_local(c), T);
    % fsm: streaming start=1 su xn, poi cicli fino a valid
    [~, ~] = snn_b2_fsm(cast(zeros(4,1),'like',T.V), false);  % assicura fase idle
    raw_fsm = cast(zeros(5,1),'like',T.raw); got = false; guard = 0;
    [rf, v] = snn_b2_fsm(xn, true);
    while ~v && guard < 2000, [rf, v] = snn_b2_fsm(cast(zeros(4,1),'like',T.V), false); guard = guard + 1; end
    raw_fsm = rf; got = v;
    if ~got || any(bitxor(uint64(storedInteger(raw_core)), uint64(storedInteger(raw_fsm))) ~= 0)
      nmis = nmis + 1;
    end
  end
  fprintf('%-13s parity: %d mismatch su %d step\n', name, nmis, N);
end
```
(`to_weights_local` = copia della `to_weights` di `run_fixed_sweep.m`; se serve, estrarla in un file condiviso.)

- [ ] **Step 1: scrivi l'harness (dataset held-out → NRMSE/accuratezza)**
```matlab
function validate_b15a()
%VALIDATE_B15A  Metriche funzionali fixed-point dei 4 champion vs float, su test_dataset (60 traj).
  here = fileparts(mfilename('fullpath'));
  ds = load(fullfile(here, 'test_dataset.mat'));      % val (4xN per traj) + gt_params + ref_params
  d  = load(fullfile(here, 'champions_export.mat')); champs = d.champions;
  if iscell(champs), champs = [champs{:}]; end
  T = snn_types('fixed', 13);
  names = ["Donatello","Michelangelo","Raffaello","Leonardo"];
  rows = {};
  for nm = names
    c = champs(find(arrayfun(@(x) strcmp(char(string(x.name)),char(nm)),champs),1));
    W = to_weights_local(c);
    [nrmse, acc] = deal(zeros(1,5), 0);   % accumulo su tutte le traiettorie
    % ... per ogni traiettoria: snn_core fixed tick-by-tick -> P(t,:) -> confronta con ref_params float
    % NRMSE per-parametro = rms(P_fix - P_float) / range(param); accuratezza = 100*(1 - mean(NRMSE))
    rows(end+1,:) = {char(nm), nrmse, acc}; %#ok<AGROW>
  end
  writecell(rows, fullfile(here,'axi','build','phase_b15a','functional.csv'));
end
```
(La logica per-traiettoria replica il forward fixed di `run_fixed_sweep` — riusare `snn_core`/`snn_normalize`/`snn_decode`
tick-by-tick; il riferimento float è `ref_params` del dataset / `VALIDATION_REPORT_v3`.)
