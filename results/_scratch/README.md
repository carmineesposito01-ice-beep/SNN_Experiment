# results/_scratch — run esplorativi ("spuri")

Risultati di run **esplorativi / di conferma** (non parte dello studio principale), prodotti via
`scripts/scout.sh ... --tag <TAG>`. Vengono pushati qui per essere recuperabili (git pull) senza
inquinare `results/EventProp_Study/`.

Convenzione tag: `SCOUT_<descrizione>` o `<STUDIO>_<test>` (es. `SR_NOSR`, `MARGIN_l01`).
Sono cancellabili in qualsiasi momento: nessuno ci si appoggia.
