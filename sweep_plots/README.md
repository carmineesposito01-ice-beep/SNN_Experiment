# sweep_plots/ — Grafici di sweep

Archivio di grafici prodotti durante gli **sweep** di iperparametri/architettura (aggregati e
confronti tra run). Materiale di consultazione.

I dati grezzi degli sweep sono in `results/` (es. `SW_OptimizerSweep/`, `EventProp_BigSweep*/`) e i
riepiloghi CSV nella root (`_sweep_master.csv`, `sweep_summary_*.csv`); gli script relativi sono
`sweep_optimizers.py` (root) e i notebook `*_Sweep.ipynb`.
